# Python Object S2

Ce dépôt contient l'ensemble des travaux pratiques (TP) réalisés en **Python** durant le **deuxième semestre (S2)** de la **1ère année Licence Informatique** en Tunisie.

## Objectif du dépôt

L'objectif de ce dépôt est de regrouper tous les TP réalisés dans le module Python du semestre 2, afin de :

- Faciliter la révision des concepts étudiés
- Garder une trace des solutions développées
- Servir de support d'apprentissage pour les étudiants
- Partager des exemples pratiques liés à la programmation orientée objet en Python

## Contenu

Le dépôt contient plusieurs dossiers, chacun correspondant à un TP du semestre.

Chaque TP peut inclure :

- Des exercices de programmation en Python
- Des exemples d'utilisation des concepts orientés objet
- Des scripts Python (.py)
- Des corrections ou solutions proposées
